# Printing text

### PROBLEM

The given program outputs "__C++__".
Change the code to output each character on a new line, resulting in:
__C__<br>
__+__<br>
__+__<br>

### CODE
```cpp
#include <iostream>
using namespace std;

int main() {
  	//modify given statement
	  cout << "C\n+\n+";
  	return 0;
}
```

### OUTPUT OF THE CODE
> C<br>
> +<br>
> +<br>
