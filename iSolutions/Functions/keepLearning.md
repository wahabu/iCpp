# Intoduction to functions

#### Task

Complete the function in order to print the sentence "Keep learning" and call it from main() function.

### CODE

```cpp
void printSentence() {
    //complete the function
    cout << "Keep learning!" << endl;

}

int main() {
    //function call
    printSentence();
    return 0;
}
```
