# Printing text

### PROBLEM

The given program outputs "C++".
Change the code to output each character on a new line, resulting in:
C
+
+

### CODE
```cpp
#include <iostream>
using namespace std;

int main() {
  	//modify given statement
	  cout << "C\n+\n+";
  	return 0;
}
```
