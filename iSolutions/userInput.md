# User input

### PROBLEM

Write a program that takes two numbers as input, assigns those values to the provided a and b variables, then outputs their sum. 

### CODE
```cpp
#include <iostream>
using namespace std;

int main() {
    int a;
    int b;
    // your code goes here
    int sum;

    cin >> a >> b;
    sum = a + b;
    cout << sum << "\n";

    return 0;
}
```

### OUTPUT OF THE CODE
#### Sample SInput
> 137<br>
> 53
#### Sample Output
> 190<br>
